# **WonderKart**

**WonderKart is a responsive e-commerce web application designed to provide a seamless product browsing experience. This project was developed as a task-based skill demonstration to showcase front-end web development capabilities, including HTML, CSS, and interactive image handling.**

### **Key Features**

* **User-friendly interface for browsing various products.**
* **Optimized image gallery for a visually engaging shopping experience.**
* **Clean and structured codebase for easy navigation and maintenance.**

### **Technologies Used**

* **HTML5**
* **CSS3**
* **Javascript**
* **Firebase**
